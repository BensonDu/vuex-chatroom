## 项目初始化
``` bash
# 克隆项目代码
git clone git@github.com:BensonDu/vuex-chatroom.git

# 进入到项目根目录
cd vuex-chatroom

# 安装
npm install

# Serve with hot reload at localhost:8080
npm run dev

#测试地址
http://localhost:8080

```
